# Skills

Curated OpenClaw skills from my local environment, organized for sharing and version control.

Repository layout:

- `skills/` contains the skill folders
- each skill keeps its own `SKILL.md`, scripts, references, and metadata
- original author/license information is preserved inside the copied skill content where available

Included skills:

- `openclaw-config-repair` - diagnose and repair local OpenClaw config/runtime issues
- `resource-hunter` - search public pan links, torrents, magnets, and public video URLs
- `win-desktop` - Windows desktop control for mixed WSL/Windows workflows
- `wsl-winbridge` - WSL to Windows bridge automation and desktop control
- `vision-reader` - image understanding and OCR-oriented vision skill
- `proactive-agent` - proactive agent workflow patterns
- `youtube-watcher` - fetch and summarize YouTube transcripts
- `find-skills` - discover and install skills from the ecosystem
- `memory-manager` - local memory management and snapshot/search tooling
- `skill-vetter` - security-first vetting before installing skills
- `auto-updater` - automatic OpenClaw/skill update workflow
- `chaoxing-homework` - Chaoxing homework automation workflow
- `self-improving` - capture corrections and iterative improvements
- `ontology` - typed knowledge graph / structured memory skill
- `multi-search-engine` - multi-engine search integration
- `humanizer` - rewrite text to sound more natural and human-written
- `github` - GitHub operations via `gh`

Notes:

- This repo is a curated collection, not a claim of authorship for every skill.
- Some skills were authored by third parties and are preserved here with their existing metadata.
- Runtime caches such as `__pycache__` and `.pytest_cache` are excluded.
