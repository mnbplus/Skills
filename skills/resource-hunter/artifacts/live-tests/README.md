# Live Test Artifacts

This directory contains probe-time evidence and temporary research material collected during source expansion.

## What should be considered durable evidence

Keep only files that help explain or reproduce an important retrieval decision, for example:

- `next-batch/probe-summary.md`
- `the-merry-widow-1952.json`
- `inspect_the_merry_widow_1952.py`
- selected HTML/JS captures that document why a source was or was not integrated

## What should be considered temporary probe material

Many files here are one-off probe scripts or raw page captures used during investigation.
They are useful for local research, but they are **not** the primary public contract of the skill.

Examples include:
- ad hoc probe scripts
- raw search-result page captures
- downloaded frontend bundles used only for reverse engineering

## Sync guidance

Before external sync/review, treat this directory as:
- evidence/supporting material
- not the main runtime deliverable
- something that may be trimmed further if a cleaner sync package is needed
